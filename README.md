# System rejestracji wizyt w placówce medycznej

Projekt zaliczeniowy: prosta aplikacja webowa w Pythonie do rejestrowania
wizyt pacjentów u lekarzy.

🚧 **Projekt w budowie** — poszczególne elementy (backend, frontend, testy,
skrypty pomocnicze, CI oraz agent AI) dodawane są stopniowo, każdy na
osobnej gałęzi i przez Pull Request.
